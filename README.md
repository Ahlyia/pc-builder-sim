# PC-Builder Sim
PC-Builder sim is a basic browser game where you build computers, and sell them to loving clients. In this game you rack up money to increase storage sizes, and buy new parts to increase your profits, and user-base.

# Extra licensing and hosting info
I allow established web-game portals and arcade sites to host unmodified versions of this game, including ad-supported hosting, as long as:

the game remains free to play
no paywalls or paid gameplay advantages are added
no microtransactions are added
proper credit is preserved
the source/repository link is included where practical

Selling the game, relicensing it, adding monetization systems, or distributing modified commercial versions is not permitted without explicit permission.
